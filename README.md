# Gwent memory card game
[Play](https://gwent-memory-card.vercel.app/)\
A memory card game inspired by Gwent.
## Image & Sound
* Card images are fetched from the [Gwent Database](https://gwent.one/) API.
* Card back designs are from the official [Gwent game](https://www.playgwent.com/en)
* Sound and other assets are taken from _The Witcher 3_

## Cache
Card data is fetched from Gwent Database API, then stored in localStorage. Images are fetched at game start and stored locally in IndexedDB.
